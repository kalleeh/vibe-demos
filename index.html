— The Studio

We make small things, well. Each piece is a complete object — typeset with care, animated with restraint, shipped in hours rather than weeks.

01 / Practice

An open-ended exercise in editorial web craft. Each work begins as a sentence and ends as a published URL.

02 / Method

Live-coded conversations with a language model, drafted in plain text, refined under hand. No frameworks unless asked.

03 / Output

One-page websites, micro-tools, and interactive notebooks — all hosted as plain HTML, free to view, free to fork.
