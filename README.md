# vibe-demos

Quick interactive demos built live with Claude.

## Live demos

- [🇸🇪 스웨덴 미식 가이드 — Korean Swedish Food Guide](./sweden-food-guide/) — interactive travel guide with restaurant filters, modal details, dish tabs, 5-day itinerary, phrasebook with TTS, quiz, and budget calculator.
- [가스의 여정 — A Molecule's Journey](./molecule-journey/) — Three.js scrollytelling that follows a single methane molecule from an Incheon LNG terminal to a Seoul kitchen flame, in six chapters.
- [Live Globe — Seoul ⇄ Stockholm](./live-globe/) — interactive 3D Earth with both cities pinned. Live local time, weather, sunrise/sunset, real-world day/night terminator. Click a city for an AI-written "right now" snapshot in Korean + English (canned-first, key-optional).
- [한방 진료 동반자 — Intake Companion](./intake-companion/) — Korean traditional medicine intake assistant. Speak or paste symptoms in Korean; receive a structured 변증 / 처방 / 경혈 brief with 3-way model toggle (Opus 4.7 / Sonnet 4.6 / Haiku 4.5), KO↔EN, canned-first/live-optional.
- [MBTI 16유형 — Korean MBTI Read](./korean-mbti/) — short 12-question test with Korean MBTI culture nicknames (잔망 루피, 곰돌이 푸, 인싸 끝판왕…), plus an AI deep-read mode where Claude infers your type from a free-form Korean text passage. PWA, canned-first/live-optional, 3-way model toggle.
- [Resonans — A Sketchbook of Standing Waves](./resonans/) — calm tap-in-rhythm game on a hand-drawn cream paper string. Real 1D wave physics; pick a mode (n=1..5) and tap until the line inks itself in and a soft sine tone rises. No score, no timer.
- [한방병원 행정 도구함 — Clinic Admin Toolkit](./clinic-admin/) — 6-tab Excel-first toolkit for a Korean Traditional Medicine hospital administrator. Upload → transform → download for KCD-8→KCD-9 migration, 자보 한방 수가 견적, 연말정산 의료비 자료, HIRA 비급여 공개, 의무기록 보존 감사, and unified code search. All processing happens in the browser — no patient data leaves the page.

Live at: https://kalleeh.github.io/vibe-demos/
