# vibe-demos

Quick interactive demos built live with Claude.

## Live demos

- [🇸🇪 스웨덴 미식 가이드 — Korean Swedish Food Guide](./sweden-food-guide/) — interactive travel guide with restaurant filters, modal details, dish tabs, 5-day itinerary, phrasebook with TTS, quiz, and budget calculator.
- [가스의 여정 — A Molecule's Journey](./molecule-journey/) — Three.js scrollytelling that follows a single methane molecule from an Incheon LNG terminal to a Seoul kitchen flame, in six chapters.
- [Live Globe — Seoul ⇄ Stockholm](./live-globe/) — interactive 3D Earth with both cities pinned. Live local time, weather, sunrise/sunset, real-world day/night terminator. Click a city for an AI-written "right now" snapshot in Korean + English (canned-first, key-optional).
- [한방 진료 동반자 — Intake Companion](./intake-companion/) — Korean traditional medicine intake assistant. Speak or paste symptoms in Korean; receive a structured 변증 / 처방 / 경혈 brief with 3-way model toggle (Opus 4.7 / Sonnet 4.6 / Haiku 4.5), KO↔EN, canned-first/live-optional.
- [MBTI 16유형 — Korean MBTI Read](./korean-mbti/) — short 12-question test with Korean MBTI culture nicknames (잔망 루피, 곰돌이 푸, 인싸 끝판왕…), plus an AI deep-read mode where Claude infers your type from a free-form Korean text passage. PWA, canned-first/live-optional, 3-way model toggle.
- [Resonans — A Sketchbook of Standing Waves](./resonans/) — calm tap-in-rhythm game on a hand-drawn cream paper string. Real 1D wave physics; pick a mode (n=1..5) and tap until the line inks itself in and a soft sine tone rises. No score, no timer.
- [Sketchwings — A Glide Across Pencil Hills](./tinywings/) — pencil-bird glides across rolling cream-paper hills in the visual grammar of resonans. Tap on the downslope to dive, release on the upslope to launch. Procedural sum-of-sines terrain (deterministic per-day seed), boost meter that recharges on airtime, and a slow day-into-night palette cycle (dawn → noon → dusk → night) that ends the run when night fully falls. localStorage best distance, perfect-launch counter, no backend.
- [A New Hope — A Vibe-Demos Homage](./starwars-homage/) — 42-second cinematic homage in four beats: opening crawl, Tatooine binary sunset, X-wing launch over Yavin, Death Star trench run, then the title card. Authored as a HyperFrames composition (HTML + GSAP + CSS, no Three.js), rendered to a single MP4 and served from a thin PWA shell with poster + play-to-fullscreen. Original orchestral cue (not John Williams).
- [한방병원 행정 도구함 — Clinic Admin Toolkit](./clinic-admin/) — coherent 10-section admin app for a Korean Traditional Medicine hospital 행정원장. Left-rail nav grouped into 시작 / 청구·코드 / 보고·감사 / 조직·인증; persistent topbar with D-day chip, sync LED, **▶ 시연** pill (re-opens the welcome tour), and **⌘K** command palette (fuzzy search across tabs, KCD codes, saved 자보 cases, licenses, accred items). First-run welcome overlay with one-click "샘플 데이터로 둘러보기" that seeds every tab simultaneously. Toast notifications surface cross-tab activity. Mobile-first touches: hamburger drawer, **camera capture + on-device OCR** for license cards (Tesseract.js · 한+영, auto-extracts 만료일), **음성 입력** dictation in the AI tab (ko-KR), **공유** buttons that use Web Share API with clipboard fallback, and haptics on save/delete. Tools: **00 오늘** dashboard (D-days, resume cards, activity feed, insight tiles, .ics export), **01–06** KCD-8 정비 / 자보 EDI 정산 / 연말정산 사전검증 / HIRA 비급여 반기보고 / 의무기록 보존 감사 / 통합검색, **07 AI 코딩 어시스트** (BYO key, Opus 4.7 / Sonnet 4.6 / Haiku 4.5), **08 면허·자격 갱신 트래커** with photo+OCR and .ics, **09 인증평가 체크리스트** with print-to-PDF. Cross-tab live sync via BroadcastChannel + localStorage; image attachments live in IndexedDB; everything runs in the browser, no backend, no patient data leaves the page. **PWA install help** detects standalone vs browser mode, shows a rail-foot **↓ 홈 화면에 추가** button + first-visit nudge, and opens a hand-sketched walkthrough modal with iOS Safari / Android Chrome / Desktop tabs (UA-aware default) — uses `beforeinstallprompt` for native one-tap install where available, manual steps where not.

Live at: https://kalleeh.github.io/vibe-demos/
